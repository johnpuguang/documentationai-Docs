## Overview

Welcome to the John Chen documentation platform. You can organize, manage, and share all your project documentation in one centralized space. This platform supports rich MDX content, interactive components, and seamless customization to match your brand, including the primary color `#428BCA`.

Use this space to create guides, API references, changelogs, and FAQs. It scales with your projects, from simple notes to comprehensive developer hubs.

## Quick Start Guide

Get up and running in minutes.

<Steps>
  <Step title="Create Your First Page" icon="file-plus">
    Navigate to the content folder and add a new `.mdx` file with YAML frontmatter.
  </Step>
  <Step title="Add Components" icon="puzzle">
    Use available components like `<Card>` and `<Steps>` to structure your content.
  </Step>
  <Step title="Customize Theme" icon="settings">
    Update the theme config with your brand color `#428BCA`.
  </Step>
  <Step title="Preview and Deploy" icon="rocket">
    Run the dev server and deploy to production.
  </Step>
</Steps>

<CodeGroup tabs="npm,yarn">
````bash
npm install your-docs-package
````

````bash
yarn add your-docs-package
````
</CodeGroup>

## Customization Options

<Expandable title="Brand Color and Theme Setup" default-open="true">

Configure your documentation to match John Chen branding.

```javascript
// theme.config.js
export default {
  colors: {
    primary: '#428BCA',
  },
  logo: '/john-chen-logo.png',
};
```

Update the `{primary}` color to `#428BCA` for headers, buttons, and links. Add custom fonts or icons as needed.
